## ⚙️ How to Fix Setup Failures
If the installer stops or shows an error, perform these manual clean-up steps.

### Step 1: Remove Old Files
If you tried to install PicPick before, old files might block the new version. Open your File Explorer. Navigate to `C:\Program Files (x86)\` and check for any existing PicPick folder. Delete the folder if it exists.

### Step 2: Clear Temporary Data
Windows keeps temporary files that can corrupt new installations. Press the Windows Key + R on your keyboard. Type `%temp%` in the box and press Enter. A folder will open. Select all files in this folder and press the Delete key. Skip any files that the system says are "in use."

### Step 3: Manage Security Settings
Sometimes Windows Defender blocks legitimate software. Temporarily disable "Real-time protection" in your Virus & threat protection settings. Run the installer again. Once the installation finishes, turn your security settings back on.
